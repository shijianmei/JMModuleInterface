# JMModuleInterface

一个模块化服务层

## Requirements
iOS 11.0
## Installation
To install
it, simply add the following line to your Podfile:

```ruby
source 'git@github.com:shijianmei/JMPrivateRepos.git'
pod 'JMModuleInterface'
```
